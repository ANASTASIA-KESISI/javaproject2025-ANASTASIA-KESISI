
### How to run the game
1. Clone this repository
2. Open cmd
3. Change the current directory to the game's folder
   ```sh
   cd C:\...{YOUR PATH TO THE GAME FOLDER}...\javaproject2025-ANASTASIA-KESISI\Assignment_4\src
   ```
4. Compile the java classes
   ```sh
   javac *.java
   ```
5. Run Main.java
   ```js
   java Main
   ```
6. Enjoy!
