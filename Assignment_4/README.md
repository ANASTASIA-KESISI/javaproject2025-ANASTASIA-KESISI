
### How to run the game
1. Open cmd
2. Change the current directory to the game's folder
   ```sh
   cd C:\...{YOUR PATH TO THE GAME FOLDER}...\javaproject2025-ANASTASIA-KESISI\Assignment_4\src
   ```
3. Compile the java classes
   ```sh
   javac *.java
   ```
4. Run Main.java
   ```js
   java Main
   ```
5. Enjoy!
